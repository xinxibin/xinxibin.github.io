在苹果MAC系统上给树莓派SD卡装系统raspbian

发表于 2017-06-08 | 分类于 Document ， Raspberry | 阅读次数

树莓派折腾

新买一个 Raspberry Pi 3B，淘宝购买，想回来实现，只能设备的控制，使用Siri来控制。下面就是树莓派系统安装到 SD卡的教程

下载系统

系统下载 一个带图形一个不带

工具

  • SD卡格式化工具 SDFormatter
  • Mac给sd卡装系统软件 etcher
  • 先用 SDFormatter 选择你的SD卡 格式化，之后在用Etcher
  • 选择img系统开始刷等待完成

系统烧到SD卡完成之后

  • 插入树莓派，有显示器的链接显示器又启动画面，开机之后可以在右上角找到直接链接无线网，鼠标放上去别动显示ip地址
  • 没有显示器的等一会，查看树莓派ip地址插入有线，在路由器中获取
  • 默认不开启ssh访问
    • 在根目录创建一个 ssh 文件夹（无显示器）（把SD卡查到电脑上，穿件一个文件名为 ssh 的文件，不要有任何后缀如下图）
    • 左上角第一个树莓的图标点开—> Preferences —> Raspberry Pi Configuration —> 第二个选项Interface 里面的ssh 打开保存退出重启
    • 输入sudo raspi-config终端窗口;
      选择 Interfacing Options;
      导航到并选择 SSH;
      选择 Yes;
      选择 Ok;
      选择 Finish;

已开通ssh 登录方式

  • ssh pi@x.x.x.x （x.x.x.x 为树莓派ip地址）
  • 接着让输入密码，默认密码为 raspberry ，yes 之后，进入pi
