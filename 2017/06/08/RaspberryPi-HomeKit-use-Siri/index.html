RaspberryPi-HomeKit use Siri

发表于 2017-06-08 | 分类于 Document ， Raspberry | 阅读次数

树莓派系统安装可以参考RaspBerryPi

下面就是安装 Homebridge 使用家庭App，发现桥，使用米家 Yeelisht 彩光灯

  • 树莓派安装 Nodejs、树莓派安装 Homebridge；米家 YeeLight 彩光灯接入 Apple HomeKit

  • sudo apt-get update

  • sudo apt-get upgrade

  • 安装nodejs curl -sL https://deb.nodesource.com/setup_7.x | sudo -E bash -

  • apt-get install nodejs 切换到root 用户安装执行 sudo su root

  • sudo apt-get install libavahi-compat-libdnssd-dev pi用户下

  • 安装 Homebridge及相关依赖



  • 安装 Homebridge-Yeelight sudo npm install -g homebridge-yeelight –registry=http://r.cnpmjs.org

homebridge 运行没有在手机家庭中发现，
我改了下树莓派链接的网络，重启树莓派，手机接入同一个网络后 ，出现了

  • 安装 screen apt-get install screen
  • 开机启动 网友说不稳定


保存重启树莓派 sudo reboot
