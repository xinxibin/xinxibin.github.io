发表于 2016-07-18   |   分类于 Code ， iOS   |     |   阅读次数

CAShapeLayer类

CAShapeLayer 继承 CALayer，可以使用 CALayer 的所有属性和方法 ；

它依附于一个给定的path,必须给与path,而且,即使path不完整也会自动首尾相接；

strokeStart以及strokeEnd代表着在这个path中所占用的百分比；

CAShapeLayer动画仅仅限于沿着边缘的动画效果,它实现不了填充效果。
